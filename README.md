# TALES

Platform to manage a tales competition.

## Requirements

- Vagrant

## Installation

- Run `vagrant up`
- Run `vagrant ssh`
- Run `rvm install ruby-2.2.2`
- Run `gem install rails`
- Run `cd /vagrant/Tales`
- Run `bundle install`
- Run `rails server -b 0.0.0.0`
- Open in your browser http://192.168.33.10:3000
